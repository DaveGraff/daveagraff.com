How was this all made?

That's a great question!

For starters, I knew that I didn't want to spend too much money on this, so I decided I would use a free service to host it. I ended up with PythonAnywhere, since they supported the framework I was used to (Flask). This comes with a few drawbacks, though. The most important of which is storage space. Due to that, I've taken a few measures to decrease the overall size of what needs to be stored:
  • All images on this site are hosted in my Google Drive
  • Information on my projects is pulled dynamically from GitHub
  • The majority of this content is HTML and CSS
Thanks to all of that, I'm only paying for my domain name, which simply points to PythonAnywhere.

Total cost: $12/Year
